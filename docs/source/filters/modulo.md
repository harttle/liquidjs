---
title: modulo
---

{% since %}v1.9.1{% endsince %}

Returns the remainder of a division operation.

Input
```liquid
{{ 3 | modulo: 2 }}
```

Output
```text
1
```

Input
```liquid
{{ 24 | modulo: 7 }}
```

Output
```text
3
```

Input
```liquid
{{ 183.357 | modulo: 12 }}
```

Output
```text
3.3569999999999993
```
